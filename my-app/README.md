# UserList - Perfis fictícios com Next.js

Projeto desenvolvido para prática de Front-End com Next.js, consumindo a API Random User Generator para exibição de perfis fictícios.

## Funcionalidades
- Listagem de usuários obtidos da API
- Página de detalhes de cada usuário
- Página estática "Sobre o Projeto"
- Componentização com Header, Footer, UserCard, ActionButton e InfoCard

## Tecnologias
- Next.js
- Tailwind CSS
- Random User Generator API

## Como executar
1. Instale as dependências:
   npm install

2. Execute o projeto:
   npm run dev

3. Acesse:
   http://localhost:3000

## API utilizada
https://randomuser.me/api/?results=10&seed=preparacao-nex
